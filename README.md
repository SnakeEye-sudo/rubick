## About Rubick

Open-source plugin-based desktop efficiency toolbox. Lightweight, powerful, and highly customizable.

## Downloads

- [Rubick Mac OS](https://github.com/rubickCenter/rubick/releases)
- [Rubick Windows](https://github.com/rubickCenter/rubick/releases)

## Docs

- [Rubick Website](https://rubick.vip)
- [Rubick Docs](https://rubickCenter.github.io/docs/)

## Features

- Plugin management based on npm
- WebDAV multi-device sync
- System plugin integration
- Quick app & file launch
- Enterprise deployment
- English & Hindi support
